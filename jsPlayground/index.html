JS is EVENT BASED

For example:
  • onclick - mouse click
  • ondblclick - mouse double click
  • onmouseover - mouse moves over
  • onmouseout - mouse leaves
  • onload - the element has been drawn
  • onchange - form element VALUE changes
  • onfocus - form element gains focus
  • onblur - form element loses focus
  • and many others
